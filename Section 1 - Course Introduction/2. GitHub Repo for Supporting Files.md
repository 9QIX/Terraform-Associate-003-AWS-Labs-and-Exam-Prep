# GitHub Repository for Supporting Files

## Overview

All course materials, lab files, and code samples are available in the official GitHub repository.

## Repository Details

- **Repository URL**: [https://github.com/btkrausen/hashicorp](https://github.com/btkrausen/hashicorp)
- **Course Files Location**: Navigate to the `Terraform` directory

## What's Included

- Complete lab code for all 70+ hands-on exercises
- Downloadable PDF guides for each lab
- Configuration files and examples
- Prerequisites and setup instructions
- Code samples organized by exam objective

## How to Use

1. Clone or download the repository to your local machine
2. Navigate to the `Terraform` directory
3. Locate the specific lab folder corresponding to the course section
4. Follow along with the video lectures using the provided code

## Best Practices

- Review the code before starting each lab
- Use the repository as a reference if you skip sections
- Keep your local copy updated for any course corrections or additions
